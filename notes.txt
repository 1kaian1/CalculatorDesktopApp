Práce s GITem:

git fetch	   Stáhne nové commity ze vzdáleného repozitáře, ale neaktualizuje pracovní kopii.
git pull	   Stáhne nové commity a automaticky je sloučí do aktuální větve.
git commit     Uloží změny do lokálního repozitáře. Neodesílá je na vzdálený server!
git push       Odešle commity z lokálního repozitáře na remote (GitHub, GitLab...).
git merge      Kombinuje změny ze dvou větví do jedné.
